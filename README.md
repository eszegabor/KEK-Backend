# KEK-Backend

Base repo:
https://github.com/nitslaszlo/jedlik-express-mongoose-ts-backend-template-session

Backend deployment
https://kek-backend.onrender.com

Backen deployment lépései a render.com-on:
- Pullrequest(PR)-Merge utasítás előtt:
1. saját repository dist mappa törlése
2. "npm run postinstall" parancs futtatása
3. PR előtti commit a saját_repóba
4. Pullrequest saját_repo -> main
5. Merge
